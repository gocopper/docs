# React + Tailwind

If you'd like to build your frontend with [React](https://reactjs.org/) and style it with [Tailwind](https://tailwindcss.com/), use the `vite:react:tailwind` frontend stack -

```bash
$ copper create -frontend=vite:react:tailwind github.com/nasa/starship
```

Run the server -

```bash
$ cd starship
$ copper run -watch
```

In a separate terminal window, run the Vite server -

```
$ cd starship/web
$ npm run dev
```

Open [_http://localhost:5901_](http://localhost:5901)
